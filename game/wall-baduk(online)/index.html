Wall-baduk (벽바둑)

대기 중…
현재 턴:
P1(빨강): 0 / P2(파랑): 0
게임 설명
  • 목표: 벽으로 칸들을 막아 내 말만 들어 있는 영역을 많이 만들기.
  • 턴 진행: 내 말 하나 선택 → 이동(건너뛰기 가능) → 벽 1개 설치 → 상대 턴.
  • 이동 규칙:
    • 상하좌우로 1칸 또는 1칸+1칸(직각 이동 가능).
    • 벽은 지나갈 수 없음, 다른 말이 있는 칸으로는 이동 불가.
    • 이동 가능한 칸은 짙은 회색으로 표시.
    • 이동할 곳이 없으면 바로 벽 설치로 진행.
  • 벽 설치: 빈 가로/세로 벽 칸에만 설치 가능, 한 번 놓으면 되돌릴 수 없음.
  • 영역 & 승리:
    • 벽으로 서로 이어진 빈 칸들의 묶음을 영역이라고 함.
    • 한쪽 플레이어의 말만 들어 있는 영역만 그 플레이어가 가짐.
    • 두 플레이어의 말이 함께 있으면 그 영역은 아무도 가지지 않음.
    • 섞여 있는 영역이 모두 사라지는 순간 게임 종료 → 자신의 영역에 포함된 칸 수가 더 많은 사람이 승리.
