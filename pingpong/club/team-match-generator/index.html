탁구 단체전 생성기

참석자 목록

🏓 사용법 안내

1. 기본 사용

  • ➕ 이름, 부수 입력 → 선수 추가
  • ⭐ 팀장 후보(1~6부) 선택 → 팀장 선정 완료 (선택 안 해도 됨)
  • 👆 참석자 클릭 시 파란색 → 선택
  • 🗑️ 추가 선수 전체 삭제 가능

2. 팀 배정

  • 자동 배정
    • - 팀장 O → 각 팀에 1명씩 고정
    • - 팀장 X → 팝업에서 팀 수 입력
    • - 부수 합 균등, 동일 부수는 무작위 배치
  • 수동 배정
    • - 팀 수 입력(최대 8팀)
    • - 참가자 버튼 클릭 → 팀에 직접 배정
    • - 팀 카드: ↔ 교체 / → 이동 / ✖ 해제 / + 선수 추가

3. 경기 시작

  • 🚀 팀 확정 후 경기 시작 → Firebase 저장 & 경기 진행 화면 이동

4. 기타 기능

  • 🖼️ 팀 이미지 저장: PNG로 저장
  • 🔄 전체 초기화: 모든 데이터 초기화
  • 📱 신규 선수 = 노란색 강조, 모바일 대응
