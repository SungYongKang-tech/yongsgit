참석자 목록

📘 사용법 안내

  • 선수 추가: 이름과 부수를 입력 후 '선수 추가' 클릭
  • 팀장 선정: 1~6부 선수를 클릭해 후보로 등록 → '팀장 선정 완료' 클릭
  • 참가자 선택: 클릭해서 파란색(선택됨)으로 만들기
  • 팀 자동 배정: '팀 자동 배정' 클릭 시 부수 균등하게 자동 팀 구성
  • 경기 시작: 서버에 팀 구성 정보 저장 후 경기 페이지로 이동
  • 팀 이미지 저장: 구성된 팀 정보를 이미지로 저장
