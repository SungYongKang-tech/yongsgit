탁구 단체전 생성기

참석자 목록

🏓 사용법 안내

1. 선수 등록

  • ➕ 이름과 부수 입력 → 선수 추가 버튼 클릭
  • 🗑️ 잘못 추가된 선수는 추가 선수 삭제로 일괄 제거
  • ✨ 노란색 선수 = 새로 추가된 선수 (상대팀으로 자동 분류)

2. 팀 배정

  • 자동 배정
    • - 참석자 선택 시 선택된 인원만 팀 배정
    • - 팀 수 입력 후 부수합과 인원수를 최대한 균등하게 자동 배치
  • 수동 배정
    • - 팀 수 입력(최대 8팀)
    • - 선수 옆 → 팀 버튼을 눌러 원하는 팀에 직접 배정
    • - 팀 카드 내 ↔ 교체, → 이동, ✖ 해제로 수정 가능
    • - + 버튼으로 즉석에서 팀에 새로운 선수 추가 가능

3. 교류전 생성

  • ⚔️ 우리팀 / 상대팀 / 혼합팀 팀 수를 각각 입력
  • - 혼합팀이 있으면 우리팀/상대팀에서 선수를 가져와 자동으로 구성
  • - 부수합과 인원수를 균등하게 맞추도록 자동 조정
  • - 생성 후 각 팀원 교체/이동/해제 가능

4. 경기 시작

  • 🚀 경기 시작 버튼 → Firebase에 팀 저장 → 경기 진행 화면으로 이동
  • 저장 전 팀 이미지 저장으로 PNG 파일로 기록 가능

5. 기타

  • 🔄 전체 초기화 버튼 → 화면 및 데이터 초기화
  • 📱 모바일 화면 최적화 지원

⚔️ 교류전 설정

1️⃣ 우리팀 선택

2️⃣ 상대팀 선택

3️⃣ 팀 수 설정

각 항목은 클릭 또는 직접 입력 가능합니다.
