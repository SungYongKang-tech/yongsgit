⚡ 전기작업 안전교육
원칙 1

전기작업은 반드시 유자격자만

전기기기·전로의 설치·해체·정비·점검 등 모든 전기작업은 자격·면허·경험을 갖춘 유자격자만 수행합니다.

원칙 2

작업 전 반드시 전로 차단

감전 위험이 있는 노출 충전부 또는 그 부근에서 작업 시 반드시 전로를 차단해야 합니다.

차단 → 잠금(LOTO) → 표찰(Tag) → 잔류전하 방전 → 무전압 확인 → 접지 설치
원칙 3

충전전로 작업 시 필수 조치

  • 방호·차폐·절연 조치: 신체·공구가 전로에 닿지 않도록 차폐
  • 절연용 보호구 착용: 고무절연장갑·절연장화·활선보호구 등
  • 절연 방호구 설치: 고압/특고압 필수, 저압도 접촉 우려 시 설치
  • 활선작업기구 사용: 절연봉·활선매트 등 사용
원칙 4

정전전로·인근 작업 시 안전조치

  • 접지기구 제거 여부 확인
  • 모든 작업자가 이격 상태인지 확인
  • 잠금·표찰은 설치한 사람이 직접 제거
  • 이상 유무 확인 후 전원 재투입
주의

이동·휴대전동공구 사용 주의

  • 젖은 손으로 플러그 취급 금지
  • 노출 충전부와 공구가 접촉되지 않도록 유지
  • 습윤 장소는 방수형 플러그 사용
  • 잠금장치가 있으면 반드시 잠금
안전거리

접근 한계거리 준수

충전전로에 근접할 때는 전압별 접근 한계거리를 준수해야 합니다.

울타리 설치 또는 절연 보호구로 근접을 방지합니다.

작업 절차

작업 전·중·후 안전관리

✔ 작업 시작 전
  • 작업 목적·범위·절차 확인
  • 아크·전격 등 전기 위험요인 파악
  • 작업책임자 지정
  • 보호구 준비·점검
  • 도면 및 기기정보 사전 숙지
✔ 작업 중
  • 임시운전·중단 시 조치
  • 보호구 착용 상태 상시 점검
  • 관계자 외 출입 통제
✔ 작업 후
  • 시운전
  • 이력 기록 및 인계
  • 잠금·표찰은 설치자가 직접 제거
