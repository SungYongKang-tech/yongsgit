메인화면 예시 영역

현재 사용 중인 탁구대회 웹앱 메인화면에 이 도움말 버튼과 모달만 가져다 넣으시면 됩니다. 아래 코드는 단일 파일로 동작하도록 작성되어 있어 테스트 후 필요한 부분만 기존 코드에 합치면 됩니다.

🏓 탁구대회 웹앱 사용 안내

참가자와 운영진이 쉽게 사용할 수 있도록 주요 기능과 진행 순서를 정리했습니다.

대회 운영을 더 쉽고 빠르게

이 웹앱은 대회 생성, 선수 등록, 조별리그 진행, 본선 토너먼트, 경기 결과 입력, 순위 계산, QR 공유까지 한 번에 처리할 수 있도록 구성되어 있습니다.

📋 무엇을 할 수 있나요?

  • 대회 정보와 선수 명단 등록
  • 예선 리그 및 본선 토너먼트 생성
  • 경기 결과와 세트 점수 입력
  • 순위 및 진출 결과 자동 반영

📱 어떤 점이 편리한가요?

  • 휴대폰 화면에 맞춘 큰 버튼 구성
  • 운영진이 현장에서 바로 입력 가능
  • 참가자도 링크/QR로 즉시 확인 가능
  • 종이 대진표 없이 실시간 진행 가능
처음 사용하는 경우에는 대회 생성 → 경기 생성 → 경기 진행 → 결과 확인 순서로 이용하시면 가장 쉽습니다.

🛠 대회 생성 / 명단 관리

  • 대회명, 일정, 장소 등록
  • 사업소별 선수 등록 및 수정
  • 남자/여자 구분 표시
  • 상위/하위 부수 구분 가능

🎯 경기 생성

  • 예선 리그 자동 편성
  • 본선 토너먼트 자동 생성
  • 대진표를 한눈에 확인 가능
  • 운영 방식에 맞게 구조 조정 가능

📊 결과 입력 / 순위 계산

  • 3:0 / 3:1 / 3:2 버튼식 입력
  • 승패 자동 반영
  • 세트 득실 자동 계산
  • 리그 순위 자동 재정렬

🔗 공유 / 관리자 기능

  • QR 코드 및 링크 공유
  • 잠금 기능으로 오입력 방지
  • 비밀번호 기반 수정 보호 가능
  • 운영 중 데이터 관리 편리
대회 생성 선수 등록 조별리그 토너먼트 실시간 입력 QR 공유
1

대회 정보와 참가 선수 등록

대회명, 일정, 사업소, 선수 명단을 먼저 등록합니다.

2

예선 리그와 본선 경기 생성

등록된 선수와 팀을 기준으로 조별리그 및 토너먼트를 구성합니다.

3

경기 종료 후 점수 입력

경기별 결과를 입력하면 승패와 세트 차이가 자동 반영됩니다.

4

순위 확인 및 본선 진출 반영

리그 결과에 따라 순위가 자동 정리되고 본선 대진에도 이어집니다.

🏅 조별리그 순위 산정 기준

1
승수가 높은 팀이 우선합니다.
2
승수가 같으면 세트 득실차를 비교합니다.
3
그래도 같으면 승자승 원칙을 적용합니다.
본선 토너먼트에서는 각 경기 승자가 자동으로 다음 라운드에 진출하도록 구성할 수 있습니다.

이용 시 참고해주세요

경기 전에는 선수명과 팀 구성을 먼저 확인하시고, 경기 후에는 결과를 바로 입력해 주시면 전체 진행이 훨씬 원활해집니다.

👥 참가자 안내

  • 경기 시작 전 본인 명단을 확인해 주세요.
  • 대진표와 경기 순서를 미리 확인해 주세요.
  • 결과 입력이 끝났는지 확인해 주세요.

🧑‍💼 운영진 안내

  • 대회 시작 전 명단 최종 점검
  • 경기 종료 즉시 결과 입력
  • 잠금 기능으로 실수 입력 방지

📲 접속 방법

  • QR 코드 스캔 또는 링크 접속
  • 휴대폰에서 바로 사용 가능
  • 메인화면의 도움말 버튼으로 재확인 가능

💡 추천 활용

  • 메인화면 상단에 도움말 버튼 고정
  • 카카오톡 공지문에 웹앱 링크 함께 안내
  • QR 코드와 함께 현장 배너로 활용
