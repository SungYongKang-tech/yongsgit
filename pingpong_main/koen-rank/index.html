📘 경기결과 입력 사용법

  • 1️⃣ 칩을 눌러 선수 2명(단식) 또는 4명(복식)을 선택합니다.
  • 2️⃣ A팀/B팀 자동 배정이 생성되며, A팀 승 또는 B팀 승 버튼을 눌러 결과를 저장합니다.
  • 3️⃣ 최근 경기결과는 화면 하단에서 확인·삭제할 수 있습니다.
  • 4️⃣ 통합 전적 : 월별 경기수, 승/패, 승률을 단식+복식 기준으로 한눈에 볼 수 있습니다.
  • 5️⃣ 단/복식 전적 : 모든 선수 목록이 표시되며, 각 선수의 단식/복식 전적을 따로 볼 수 있습니다.
  • 6️⃣ 개인 상세 페이지(선수 클릭) : 월별 성적, 상대/파트너 TOP, 그래프 등이 표시됩니다.
  • 7️⃣ 상세 페이지 상단 버튼으로 빠른 이동이 제공됩니다.
  • 8️⃣ 상대/파트너 분석이 자동 계산됩니다.
  • 9️⃣ 그래프가 시각화됩니다.
  • 🔟 월별 통합 기록이 카드로 표시됩니다.
