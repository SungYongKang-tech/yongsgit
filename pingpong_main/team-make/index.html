참석자 목록

🏓 KOEN 단체전 생성기 사용법

1️⃣ 선수 등록

  • ➕ 이름과 부수를 입력하고 선수 추가를 누르면 새 선수 등록
  • 🟡 새로 추가된 선수는 노란색으로 표시
  • ✖ 잘못 추가된 선수는 이름 옆 ✖ 버튼으로 개별 삭제 가능
  • 🗑️ 추가 선수 삭제 버튼으로 전체 삭제 가능

2️⃣ 팀 배정

  • 팀 자동 배정: 팀 수 선택 → 인원/부수 평균(낮을수록 강함) 균형 배정
  • 팀 수동 배정: 팀 수 입력 → 참석자 목록에서 버튼으로 직접 배정

3️⃣ 경기 시작

  • 🚀 경기 시작 버튼 클릭 시 Firebase에 팀 정보 저장

⚔️ 교류전 설정

1️⃣ 우리팀 선택

2️⃣ 상대팀 선택

3️⃣ 팀 수 설정

각 항목은 클릭 또는 직접 입력 가능합니다.
