1. server-auto-on API 추가

* 현재는 /api/server-auto-toggle 만 존재
* ON 전용 API(/api/server-auto-on) 추가 필요
* setServerAutoEnabled(true) 사용

2. 서버 자동매매 크론 수정

* 현재 09:00 자동 ON 기능 없음
* server-auto-on API 추가 후 크론 등록
* 0 9 * * 1-5 curl -s "http://localhost:3000/api/server-auto-on"

3. PM2 재시작 후 자동매매 상태 유지 확인

* serverAutoEnabled 값이 true로 유지되는지 점검
* 재부팅 후에도 자동매매 ON 상태 유지 확인

4. 자동매수/자동매도 로그 점검

* 09:05 자동매수 실행 여부
* 손절(-2.3%) 정상 동작 여부
* virtualResults 저장 여부 확인
