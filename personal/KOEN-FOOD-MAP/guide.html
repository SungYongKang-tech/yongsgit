직원 참여로 완성되는 우리만의 맛집 공유 플랫폼

📌 웹앱 소개

“오늘 뭐 먹지?” 고민을 줄여주는 KOEN 맛집 공유 플랫폼이다.

본 웹앱은 남동발전 임직원 간 맛집 정보를 공유하기 위해 제작되었다.
기본 데이터는 상생협력부에서 제공한 ‘남동 맛집 100선’을 기반으로 구성되었으며, 사용자 참여를 통해 지속적으로 추가 및 수정될 예정이다.

단순한 목록이 아닌, 실제 이용자의 경험과 의견이 반영되는 살아있는 맛집 데이터로 운영된다.

🎯 운영 목적

- 회사 주변 맛집 정보를 쉽고 빠르게 공유하기 위함
- 직원 간 추천 및 후기 기반의 신뢰도 높은 정보 제공
- 개인의 경험을 조직 전체의 데이터로 확장

참여할수록 더 좋은 정보가 쌓이는 구조로 운영된다.

👥 이용 대상

본 웹앱은 남동발전(KOEN) 임직원을 대상으로 운영되는 서비스이다.

내부 직원 간 맛집 정보 공유를 목적으로 한다.

🛠️ 주요 기능

1. 🔍 검색 기능
- 식당명, 메뉴, 주소를 기준으로 빠르게 검색 가능

2. 🏷️ 카테고리 및 태그 필터
- 음식 종류 및 특징별로 식당을 분류하여 원하는 조건으로 조회 가능

3. ⭐ 평점 기능
- 실제 이용자가 평가한 점수를 기반으로 신뢰도 높은 정보 제공

4. 💬 후기 기능
- 간단한 한줄 후기 작성 및 확인 가능

5. 📊 정렬 기능
- 평점순, 인기순 등 다양한 기준으로 정렬 가능

6. ❤️ 즐겨찾기 기능
- 자주 가는 맛집을 따로 관리 가능

✏️ 수정 및 추가 방법

메인 화면의 ‘수정요청’ 버튼을 통해 의견을 제출할 수 있다.

- 식당 정보 오류 수정 요청
- 새로운 맛집 추가 요청
- 메뉴, 주소, 태그 등 변경 요청

맛집 추가가 필요할 경우에도 별도 메뉴를 찾지 말고 수정요청 버튼에서 자유롭게 작성하면 된다.

별도의 형식 없이 자유롭게 작성하면 되며, 접수된 내용은 검토 후 반영된다.

🚀 향후 업데이트 예정 기능

본 웹앱은 지속적으로 기능이 개선될 예정이며, 다음과 같은 기능들이 추가될 계획이다.

- 📈 인기 맛집 순위 (TOP 맛집)
- 🏆 월간 추천 및 우수 맛집
- 👤 개인별 평점 및 활동 관리 기능
- 📊 데이터 기반 추천 기능
- 🔔 사용자 알림 및 참여 기능 강화

사용자 의견을 적극 반영하여 점점 더 편리한 서비스로 발전할 예정이다.

📢 이용 안내

- 자유롭게 열람 및 활용 가능
- 실제 경험 기반의 정보 공유 권장
- 적극적인 참여가 더 좋은 서비스로 이어짐

여러분의 참여가 곧 더 좋은 맛집 데이터로 이어진다.
